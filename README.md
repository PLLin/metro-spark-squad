本專案為 **2026 捷運盃黑客松**（「玩點生活・智慧串聯」創新應用競賽）參賽作品 —— **「燎原智庫・點燃同行」** 的高保真網頁版原型。

## 🚀 本地啟動與預覽步驟

若您想在本地執行本專案，您可以直接透過以下步驟啟動：

### 1. 安裝依賴與啟動開發伺服器
在專案根目錄下，開啟終端機並執行以下指令：
```bash
# 安裝相關打包依賴
npm install

# 啟動本地 Vite 開發伺服器
npm run dev
```

### 2. 電腦瀏覽器預覽
啟動後，在電腦瀏覽器中打開終端機輸出的網址（預設通常為 `http://localhost:5173`）。
您將看到**精美的電腦版雙欄介面**：
* **左側**：展示本提案的商業核心機制與即時試用流程控制台（包含 4 大演示步驟與防呆保護機制）。
* **右側**：一個完全可互動的 **iPhone 15 Pro 模擬器**，您可以在上面像操作真實 App 一樣點擊、滑動、切換頁籤。

### 3. 手機瀏覽器預覽（體驗真實 App 質感）
本原型支援手機自適應佈局。當直接使用手機瀏覽時，會**自動隱藏電腦版外框與左側控制台，直接滿版呈現手機 App 介面**，操作感與原生 App 無異。

* **步驟**：
  1. 確保您的**手機**與**電腦**連接到**同一個 Wi-Fi 網路**。
  2. 查看執行 `npm run dev` 後終端機所輸出的 `Network` 網址，在手機瀏覽器中輸入該網址，即可在手機上體驗滿版 App！

## 🎯 原型核心機制與互動測試引導

請依序點擊左側控制台的試用按鈕，或在右側手機模擬器內進行以下操作，體驗完整的點數眾籌與 AI 任務流程：

### 0. 👥 快速組成團隊 (步驟 00)
* **操作**：點擊左側控制台的 **「步驟 00：組成團隊」**（或手動於手機「金庫」頁面勾選聯絡人如「楊小芸、陳阿明」，並點選「確認組成團隊並派發 AI 任務」）。
* **現象**：隊伍將會快速建立，並自動從 `STATE.contacts` 中調用好友所攜帶的點數注入金庫。同時，AI 引擎會根據組隊人數與特徵派發專屬任務（例如：雙人週末出遊、夜市美食聚點）。
* **自訂特徵**：在組隊前，可以點擊列表各人名右邊的 **「編輯（鉛筆）圖示」** 來自由修改個人及好友的年齡與性別，觀察 AI 派發對應特徵任務的流程。

### 1. 🚦 選擇乘車隊員與模擬出站 (步驟 01)
* **操作**：此時首頁的「通勤軌跡模擬」區塊將會解鎖。在「選擇同行乘車隊員」清單中，勾選參與出行的隊友（如：我、楊小芸），並點擊下方 **「模擬乘車出站」**（或直接點選左側控制台的 **「步驟 01：模擬雙人乘車」**）。
* **防錯機制**：在組隊完成前，首頁與任務大廳皆會顯示「尚未組成團隊」的佔位卡鎖定狀態。若嘗試跨步執行，系統亦會彈出 Toast 警告並將您導航回正確步驟。
* **解鎖現象**：手機頂部的「動態島」變寬播放分析動畫，分析成功後解鎖該同行任務的商家消費步驟。

### 2. 📸 AI Vision 視覺審核 (步驟 02)
* **操作**：進入 App 內的「任務」頁籤，在步驟三點選「上傳收據或消費拍照」（或直接點選左側控制台的 **「步驟 02：模擬收據驗證」**）。
* **現象**：畫面將會進入 AI 多模態視覺模型掃描日誌，即時輸出 AI 語意識別細節，自動對照店家發票與出站軌跡。審核通過後，將發放點數獎勵（雙人同行回饋為 **`5,200 pt`**，三人同行回饋為 **`3,200 pt`**），使金庫點數**自然達標大於 15,000 pt 目標**，絕無生硬的點數注入補位。

### 3. 💰 金庫滿額分帳結算 (步驟 03)
* **操作**：當金庫眾籌點數自然累計破 `15,000 pt` 後，切換至手機【金庫】頁面，您會看見一列閃爍亮圈的提醒卡片。點選 **`立即執行金庫分帳結算`** 按鈕（或直接點選左側控制台的 **「步驟 03：金庫滿額結算」** 快速累積達標）。
* **現象**：App 內會彈出分帳解鎖視窗，顯示依人數平分後的個人取得點數（例如：15,680 pt / 3人 ➔ 每人平分得 `+5,226 pt`）。點選確認結算後，會播放領取點數特效，並安全重置系統環境，以利進行下一輪演示！
