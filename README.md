# 「燎原智庫・點燃同行」網頁版互動式原型 (Prototype)

本專案為 **2026 捷運盃黑客松**（「玩點生活・智慧串聯」創新應用競賽）參賽作品 —— **「燎原智庫・點燃同行」** 的高保真網頁版原型。

專案結合了**智慧社群金庫 (Social Vault)**、**GenAI 動態任務引擎**與 **Gemini Vision 視覺辨識驗證**。本原型旨在向評審與使用者展示完整的線下通勤、線上任務解鎖與商圈消费 O2O 點數回饋閉環。

## 🚀 本地啟動與預覽步驟

我們已經為您完成了 `npm install`。您可以直接透過以下步驟啟動並測試網頁：

### 1. 啟動開發伺服器
在專案根目錄下，開啟終端機並執行以下指令：
```bash
npm run dev
```

### 2. 電腦瀏覽器預覽
啟動後，在電腦瀏覽器中打開以下網址：
* **[http://localhost:5173](http://localhost:5173)**

您將看到**精美的電腦版雙欄介面**：
* **左側**：展示本提案的商業核心價值與快速模擬控制台。
* **右側**：一個完全可互動的 **iPhone 15 Pro 模擬器**，您可以在上面像操作真實 App 一樣點擊、滑動、切換頁籤。

### 3. 手機 Chrome 瀏覽器預覽（重點功能）
本原型支援手機自適應佈局，當使用手機瀏覽時，會**隱藏電腦版外框與側欄，直接滿版呈現 App 介面**，操作感與原生 App 無異。

* **步驟**：
  1. 確保您的**手機**與**電腦**連接到**同一個 Wi-Fi 網路**。
  2. 查看執行 `npm run dev` 後終端機所輸出的 `Network` 網址，格式通常為：
     ```text
     http://192.168.X.X:5173
     ```
  3. 在手機的 Chrome 瀏覽器中輸入該 `Network` 網址，即可在手機上體驗滿版 App！

## 🎯 原型互動測試引導

為展示 AI 任務引擎與金庫系統，請依序嘗試以下互動：

### 1. 模擬乘車出站（解鎖任務）
* **在首頁**：點選「通勤軌跡模擬」，選擇起點「台北車站」、終點「中山站」，人數選擇「雙人同行 👥」，點擊 **「模擬乘車出站」**。
* **互動現象**：
  * 手機頂部的 **「動態島 (Dynamic Island)」** 會擴張，顯示 AI 正在分析軌跡。
  * 跳出通告通知：偵測到您與小芸的雙人同行意圖，Gemini AI 已自動為您生成限時任務。
  * 下方「任務」頁籤出現**小紅點**，且網頁會自動為您切換至「任務」分頁。

### 2. 收據視覺辨識（點數金庫注入）
* **在任務分頁**：您會看到剛剛解鎖的 **「雙人週末中山站出遊」** 任務，前兩個步驟（共同出站、抵達店家）已經打勾，停留在步驟三「上傳收據或消費拍照」。
* **點擊上傳收據**（或點擊左側控制台的「模擬：上傳中山商圈收據照片」）。
* **互動現象**：
  * 畫面會進入 **Vertex AI 與 Gemini Vision 掃描動畫**。
  * 綠色的雷射掃描線上下移動，控制台會即時輸出 AI 辨識明細（識別出中山商圈合作飲料店的雙人套餐發票，比對消費時間與票證紀錄吻合）。
  * 審核成功後，動態島發出成功通知。
  * **金庫點數即時動畫增加**：點數從 **`12,480` 點動態跳動累加至 `12,730` 點**，達成率提高。
  * 成員貢獻榜中，您的個人貢獻亦即時增加，下方新增一筆入點歷史紀錄。

### 3. 金庫切換與其他功能
* **在金庫分頁**：您可以切換「家庭點數池」與「快閃戰隊金庫」模式。點擊「邀請成員加入金庫」可以輸入受邀人電話並送出模擬邀請。
* **在首頁 / 我的分頁**：點選「玉山卡綁定」，會彈出玉山公務卡連結成功視窗，完美展示企業金融卡與捷運 App 聯名合作的場景。
